## What I Build

**RegMon** — Automated regulatory monitoring for German healthcare IT. Tracks G-BA, Gematik, and KBV publications, classifies severity, and generates compliance audit trails. In production — serving a sector where missing a regulatory change isn't an option.

**Xommodo** — Tamperproof prediction ledger. An append-only, externally verifiable track record of commodity price forecasts: SHA-256 hash chain, OpenTimestamps anchors in the Bitcoin blockchain, git as off-box witness. Every forecast is committed before the outcome is known — and provably stays that way.

**WOERK** — AI-powered job application tooling. Structured generation, not template spam.

**LUZI** — Dirty-fast port scanner for exotic protocols and neglected targets. Speed, stealth, simplicity.
